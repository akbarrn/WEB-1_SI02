FAQ

Apa itu HTML?

HTML adalah bahasa markup untuk membuat halaman web.

Apa itu CSS?

CSS adalah bahasa untuk mendesain tampilan halaman web.

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman untuk membuat halaman web interaktif.

Apa itu jQuery?

jQuery adalah library JavaScript yang memudahkan manipulasi DOM.

Bagaimana cara menggunakan jQuery?

Anda dapat menyertakan jQuery melalui CDN dan mulai menulis kode jQuery di dalam tag script.
